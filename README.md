## Prerequisites

Get dcd-server from:  
https://github.com/dlang-community/DCD/releases  
Get serve-d from:  
https://github.com/Pure-D/serve-d/releases  

copy both, to example: ~/dlang-ls

add this to your .bashrc

PATH=$PATH:~/dlang-ls/

source .bashrc

# dlang support for coc

## Install

`:CocInstall coc-dlang`
